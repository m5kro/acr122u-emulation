# acr122u-emulation
Example code to show how the acr122u can be used to emulate NFC cards.

# Requirements
1. Linux
2. Python
3. pyscard
4. PCSC
5. pn533 and pn533_usb blacklisted

# How to use
Download and run emulate.py<br>
Edit TG_INIT_AS_TARGET data to specify card type
